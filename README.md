# 生态系统模拟器

这个项目是一个简单的生态系统模拟器,展示了捕食者和猎物之间的相互作用。

## 项目结构

- `index.html`: 主页面,用于显示模拟结果
- `styles.css`: 页面样式
- `script.js`: 模拟逻辑和动画

## 模拟说明

这个模拟器包含两种生物:兔子(猎物)和狼(捕食者)。它们在一个封闭的环境中相互作用:

- 兔子以草为食,繁殖速度较快
- 狼捕食兔子,繁殖速度较慢
- 如果没有足够的食物,两种生物都会死亡

模拟器展示了这两个物种数量随时间的变化,反映了真实生态系统中的动态平衡。

## 如何运行

打开`index.html`文件在浏览器中查看模拟结果。

## 进一步开发

可以考虑添加更多物种、环境因素或复杂的交互规则来扩展这个基本模型。
